# Cruise-Booking-App-iOS
Cruise booking application developed in iOS native using Swift

### HOW TO RUN:
```console
Open project in Xcode
Run on Simulator
```

### FEATURES:
1. Home screen
2. User register screen
3. User login screen
4. Search screen
5. Cruise details screen
